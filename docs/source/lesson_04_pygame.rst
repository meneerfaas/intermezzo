Les 4: games
==============================

Natuurlijk kun je met Python meer dan alleen turtletekeningen maken. In deze les ga je aan de slag met Pygame Zero om de eerste stappen te leren van game programming. Pygame Zero is een vereenvoudigde versie van de Python module `pygame <https://pypi.org/project/pygame/>`_. 

Bij het programmeren van een game komt bijna alles wat je in de vorige lessen leerde weer aan de orde. Het is belangrijk dat je de volgende technieken (enigszins) begrijpt en beheerst:
