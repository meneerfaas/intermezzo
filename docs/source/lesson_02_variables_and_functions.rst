Les 2: variabelen en functies
=============================

Vrijwel elke programmeertaal kent de volgende vijf basiselementen:

.. card-carousel:: 3

   .. card:: Variables
      :class-card: card-primary-bg

      Om data in op te slaan, bijvoorbeeld getallen of tekst.

   .. card:: Loops
      :class-card: card-primary-bg

      Om een blok code te herhalen zolang aan een bepaalde voorwaarde is voldaan.

   .. card:: Conditionals
      :class-card: card-primary-bg

      Om een blok code uit te voeren als aan een bepaalde voorwaarde is voldaan.

   .. card:: Input/Output
      :class-card: card-primary-bg

      Om data aan de gebruiker te kunnen tonen of aan de gebruiker te kunnen vragen.

   .. card:: Functions
      :class-card: card-primary-bg

      Om een blok code een naam te geven, zodat die code kan worden aangeroepen.

*Variables* en *functions* noemen we in het Nederlands *variabelen*  en *functies*, maar voor *loops*, *conditionals* en *input/output* gebruiken we meestal de Engelse benamingen.

In de vorige les heb je al veel functies gebruikt. Bijvoorbeeld :python:`turtle.forward()`, :python:`turtle.penup()` en :python:`turtle.circle()` zijn functies. In Python herken je een functie aan haakjes achter de naam.

In deze les gaan we werken met variabelen en je leert hoe je zelf functies maakt. Daarbij zullen we ook gebruik maken van de output functie :python:`print()` en de input functie :python:`input()`. Loops en conditionals bewaren we voor later.
