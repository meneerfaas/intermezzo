Les 3: coördinaten en functies
==============================

Misschien had je het niet in de gaten, maar in de vorige twee lessen heb je al veel met functies gewerkt. In je allereerste programma gebruikte je bijvoorbeeld de functie :python:`print()` om de tekst 'Hello, World!' af te drukken. Bij het turtle tekenen gebruikte je functies als :python:`turtle.forward()` en :python:`turtle.penup()` om de turtle een actie te laten uitvoeren.

De turtle module in Python bevat veel functies die je kunt aanroepen, maar soms zou je willen dat er meer functies waren. Bijvoorbeeld een functie :python:`square()` die een vierkantje voor je tekent. Of een functie :python:`triangle()`. Deze functies zijn echter niet beschikbaar. Maar geen nood, je kunt in Python zelf je eigen functies maken! In deze les leer je hoe dat werkt, nadat je hebt kennisgemaakt met het fenomeen *coördinaten*.
